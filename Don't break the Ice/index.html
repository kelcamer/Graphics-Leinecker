This game is called Don't Break The Ice!

The object of the game is to prevent the special middle square from falling.

You only get one hit per round.

If you play against the computer, it will make it's move automatically.

Press C to change the camera.
Press B to return the camera to default.
Press Y to show the instructions again.
Use the slider to change how hard you hit the ice!
